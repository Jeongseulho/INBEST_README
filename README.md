# S09P22D110 주렁주렁팀 프로젝트

## 🔍프로젝트 소개

### 📌서비스 요약

- 모의 투자 시뮬레이션 제공
- 자신에게 맞는 자산 관리 방법 제공

### 📌기획 의도

- 초보자들이 주식에 대한 두려움을 줄이고, 주식에 대해 쉽게 배울 수 있는 서비스를 제공
- 국내, 해외, 가상화폐 등 다양한 종목을 제공하여 투자의 폭을 넓힘, 금융 상품과 주식에 대한 정보를 제공하여 자산 관리 방법을 제공
